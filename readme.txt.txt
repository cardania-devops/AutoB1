== Description ==
AI Autoblogger is a WordPress plugin that generates and schedules quality, relevant, SEO friendly, blog posts using OpenAI's GPT technology.

## Features

- Automated blog post generation using OpenAI's API.
- Customizable post generation parameters, including topic, tone, frequency, and more.
- Option to include relevant external links within posts.
- Automatic selection of WordPress post categories and tags.
- User-friendly settings interface for easy configuration.
- Manual trigger option for immediate post generation.
- Scheduled auto-posting functionality.

## Installation

1. Upload the `ai-autoblogger` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings by navigating to the 'AI Autoblogger' option in the sidebar.

## Configuration

- **API Key**: Enter your OpenAI API key to enable content generation.
- **Post Frequency**: Choose how often you want the blog posts to be auto-generated.
- **Content Tone**: Define the tone of the generated content.
- **Tags**: Specify tags to tailor the content topics.
- **Automatic Post Generation**: Toggle to enable or disable automatic blog post generation.

## Usage

- To generate a post manually, navigate to the plugin's settings page and click 'Generate Post Now'.
- The plugin will automatically generate posts based on the specified frequency if auto-posting is enabled.
